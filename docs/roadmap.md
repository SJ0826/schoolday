## 목표: 수직 슬라이스 (조례 + 1교시, 약 5분)

책상에 엎드려 자다 친구가 깨움 → 교실을 걸어다님 → 이상한 시계(첫 위화감)
→ 사원증(조례 필수템) 줍기 → 1교시 종·이상 현상 → 쪽지(1교시 필수템) → "2/2 ✓" 피드백.

## 제작 단계

- [ ] **0단계** — Unity 3D 프로젝트 + 그레이박스 교실 하나
- [ ] **1단계** — 1인칭 이동 + 마우스 시점 (걸어다니기)
- [ ] **2단계** — 친구가 깨우는 오프닝 + 간단 대사
- [ ] **3단계** — 상호작용([E]) + 필수 아이템 줍기 + 획득 UI
- [ ] **4단계** — 하루 진행 매니저(조례→1교시 전환)
- [ ] **5단계** — "어긋남" 이벤트 1~2개(이상한 시계, 반복 대사)
- [ ] **6단계** — 숨기 + 몬스터 추격(후반 교시용)
- [ ] **7단계** — 엔딩 선택 화면(3분기)
- [ ] **8단계** — 조례+1교시 수직 슬라이스 완성 → 테스트
- [ ] **9단계~** — 남은 교시 확장, 어긋남 강도 상승, 스토리 살 붙이기
