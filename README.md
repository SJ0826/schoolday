# 방과 후 (schoolday)

3D 1인칭 심리 공포게임 · **Unity 6000.5.2f1 / C#**

> 아침에 엎드려 자다 친구가 깨우며 하루가 시작된다. 평범한 학교 일과처럼 보이지만
> 모든 것이 묘하게 어긋나 있다 — 나는 학생이 아니라 지친 직장인이고, 이곳은 꿈이다.
> 진실을 밝히고 **꿈에서 깨어날지, 남을지** 선택한다.

## 🎯 현재 목표 — 수직 슬라이스 (조례 + 1교시, 약 5분)

친구가 깨움 → 교실 걷기 → 이상한 시계(첫 위화감) → 사원증·쪽지 줍기 → "필수템 2/2 ✓".
진행 상황은 [Issues](../../issues)에서 추적한다.

## 🛠 개발 규칙 (요약)

- `main` 직접 커밋 금지 → 작업 브랜치(`feat/`·`fix/`·`chore/`)에서 작업.
- 작업이 끝나면 `/ship`(커밋→푸시→PR→머지), 중간 저장은 `/save`.
- Unity 생성 폴더(`Library/` 등)는 커밋하지 않음(`.gitignore`).
- 에셋 추가 전 최초 1회 `git lfs install`.
